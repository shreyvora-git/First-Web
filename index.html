MDC: Key Topics & Concepts

Module 1: Key Linear Algebra Topics

  • Matrix Rank: Finding rank using Echelon & Normal forms.
  • Solving Linear Systems: Using iterative methods like Gauss-Jacobi & Gauss-Seidel.
  • System Consistency: Analyzing Homogeneous and Non-Homogeneous equations.
  • Cryptography: Applying matrices for encoding/decoding with the Hill Cipher.

Module 2: Key Vector & Eigenvalue Topics

  • Vector Analysis:Determining Linear Dependence & Independence.
  • Core Properties: Calculating Eigenvalues and Eigenvectors.
  • Matrix Transformation: Understanding and finding a Diagonalisable Matrix.

Module 3: Key Partial Differentiation Topics

  • Core Technique: Partial differentiation of composite and implicit functions.
  • Important Proof: Euler’s Theorem for Homogeneous functions.
  • Optimization: Finding Maxima and Minima for functions of two variables.
  • Calculus Tools: Using Jacobians for variable transformation.
  • Function Approximation: Expanding functions with Taylor’s & Maclaurin’s series.

Module 4: Key Differential Equation Topics

  • Solving First-Order ODEs: Mastering Exact, Linear, and Bernoulli's equations.
  • Numerical Solutions: Approximating solutions with Euler’s method & Runge-Kutta method.
  • Engineering Applications: Applying ODEs to solve real-world problems (e.g., circuits, cooling).
